Finding your way

Suppose you encounter
a method invocation
and want to learn How It Works.


# in lib/puppet/type/file.rb
Puppet::Type.newtype(:file) do
# ...
  CREATORS = [:content, :source, :target]
  SOURCE_ONLY_CHECKSUMS = [:none, :ctime, :mtime]

  validate do
    creator_count = 0
    CREATORS.each do |param|
      creator_count += 1 if self.should(param)
    end

Look for validate
among Puppet::Type's
class methods.


# in lib/puppet/type.rb
module Puppet
class Type
# ...
  def self.validate(&block)
    define_method(:validate, &block)
  end

Same for self.should?

We're in instance scope now.


class Type
# ...
  def should(name)
    name = name.intern
    (prop = @parameters[name] and
     prop.is_a?(Puppet::Property)) ? prop.should : nil
  end
